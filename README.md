 Web-Recon-Scanner

A lightweight, educational Python tool for web reconnaissance. It gathers information about a website's security posture, SSL status, and used technologies.

 Features
- **SSL/TLS Audit:** Checks the expiration date of the SSL certificate.
- **Security Headers:** Scans for critical headers like CSP, X-Frame-Options, and HSTS.
- **Technology Stack:** Detects CMS (like Joomla/WordPress) and lists external JavaScript resources.
- **Recon:** Checks for the presence of `robots.txt`.

prerequisites
- Python 3.x
- Required libraries: `requests`, `beautifulsoup4`, `pyOpenSSL`

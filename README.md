
# Educational Python Scripts

This repository is for educational purposes only. It demonstrates how Python interacts with the Windows Registry and GUI frameworks.

## Winlocker Example
**WARNING: Do not run this script on your system.**
This script is a demonstration of how locker-type software works. It modifies registry keys and locks the screen. 
- **Purpose:** To learn how Windows Registry and `tkinter` fullscreen mode work.
- **How to disable:** Simply open Task Manager (Ctrl + Shift + Esc) and end the process.

**Note:** Always use safe environments (like Virtual Machines) when testing system-level scripts.
